📚 AI Study Planner

AI Study Planner is a web-based application that helps students plan their study schedule efficiently.
It allows users to register, log in, and generate personalized study plans.

🚀 Features

*👤 User Registration and Login

*📅 Study Planner Dashboard

*🤖 AI-based Study Plan Generation

*🔒 Secure Authentication System

*🎨 Simple and Clean UI
